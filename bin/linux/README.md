# Linux Binaries for CI/CD

These are pre-built Linux AMD64 binaries ready to use in CI pipelines.

## Contents

- `ade` - Main compiler executable
- `plugins/csharp` - C# code generator plugin
- `plugins/arch-go` - Go arch-go code generator plugin

## Usage in GitHub Actions

Copy these files to your CI repository and set up as follows:

```yaml
- name: Set up ADR enforcement tool
  run: |
    # Make binaries executable
    chmod +x tools/ade/ade
    chmod +x tools/ade/plugins/*
    
    # Install plugins to expected location
    mkdir -p ~/.ad-enforcement/plugins
    cp tools/ade/plugins/csharp ~/.ad-enforcement/plugins/
    cp tools/ade/plugins/arch-go ~/.ad-enforcement/plugins/
    
    # Add ade to PATH
    echo "${{ github.workspace }}/tools/ade" >> $GITHUB_PATH

- name: Generate architecture tests
  run: ade compile -i ./rules/my-rule.rule -p csharp -o ./generated
```

See the full documentation at: [docs/ci.md](../../docs/ci.md)

## Rebuilding

These binaries are generated by running `make build-linux` from the root of the repository.
